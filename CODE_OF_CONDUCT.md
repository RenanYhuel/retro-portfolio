# Code of Conduct

Nous adoptons le Contributor Covenant comme code de conduite. En participant à ce projet, tu acceptes de respecter ce code de conduite.

## Notre engagement

Nous souhaitons fournir un environnement d'accueil sûr et respectueux pour tout le monde.

## Nos standards

Les comportements attendus incluent :

- Respect, considération et communication constructive.
- Attitude ouverte aux retours.
- Acceptation de la diversité d'expériences et d'opinions.

Les comportements inacceptables incluent : harcèlement, insultes personnelles, discours haineux, intimidation.

## Signalement

Si tu observes un comportement inapproprié, ouvre une issue privée ou contacte les mainteneurs via les issues publiques en précisant le problème. Les mainteneurs investigueront et prendront des mesures adaptées.
