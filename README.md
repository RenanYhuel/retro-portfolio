# Retro Portfolio

Projet Next.js minimal et professionnel au style "rétro" avec des données mock fixes.

Scripts:

-   pnpm install
-   pnpm dev
-   pnpm build
-   pnpm start

Structure:

-   src/app: app router pages
-   src/components: composants UI
-   src/data: données mock
-   src/styles: CSS global
-   src/app/api/projects: API mock GET

Ce projet est intentionnellement autonome et sans base de données. Les données sont statiques dans `src/data`.
